## Deployment

This repository contains the production-ready build bundle. To deploy:

1. Upload all files to your web hosting (Node.js required for full functionality)
2. Configure environment variables for API keys and database
3. Run `node app.cjs` to start the server
4. Access the platform at your configured domain

> **Note**: This is a pre-built bundle. Backend API functionality requires a configured Node.js server with the appropriate environment variables and database setup.
